React Blog
==========

CI status: ![Build Status](https://travis-ci.org/freewind/react-blog.png?branch=master)

A demo project, uses following techniques.

1. webpack
2. babel
3. express
4. jquery
5. react
6. react-router
7. superagent

```
npm install -g babel-cli
npm install
npm start
```

Then visit <http://localhost:3000>
